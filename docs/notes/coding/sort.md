## 排序算法

### 冒泡排序
冒泡排序（Bubble Sort）会遍历若干次要排序的数列，每次遍历时都会从前往后依次的比较相邻两个数的大小；当前者数比后者大，则交换它们的位置，完成遍历。重复此操作，直到整个数列有序为止。

C语言代码实现：
```c
#define swap(a,b)    (a^=b,b^=a,a^=b)

void bubble_sort(int a[], int n)
{
    int i, j;
    int flag;         //标记

    for(i = n-1; i > 0; i--)
    {
        flag = 0;     //初始化标记

        //将最大的数放到末尾
        for(j = 0; j < i; j++)
        {
            if (a[j] > a[j+1])
            {
                swap(a[j], a[j+1]);
                flag = 1;     //发送交换，标记为1
            }
        }
        if(flag == 0)
            break;            //如果没有发生交换，则说明数列有序
    }
}

```
